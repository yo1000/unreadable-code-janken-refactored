Unreadable code
================================================================================

An exercise in refactoring unreadable code into readable code,
using “rock-paper-scissors” as the subject.


Requirements
--------------------------------------------------------------------------------

- Java 25


How to run
--------------------------------------------------------------------------------

```bash
./mvnw clean package
java -jar target/janken.jar
```


Demo
--------------------------------------------------------------------------------

```bash
java -jar target/janken.jar
1=Rock, 2=Paper, 3=Scissors... [1/2/3]: 1

Chose by You: 1=Rock
Chose by COM: 1=Rock
1=Rock, 2=Paper, 3=Scissors... [1/2/3]: 2

Chose by You: 2=Paper
Chose by COM: 1=Rock

You win!
```
